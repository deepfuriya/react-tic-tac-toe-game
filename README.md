
# Tic-Tac-Toe React Game

This project is a simple Tic-Tac-Toe game built using React. It features a clean and interactive user interface, allowing two players to play the game on the same computer. The game is developed using **React components** and the **State** hook for state management.



##

![tic-tac-toe](https://github.com/deepfuriya/react-tic-tac-toe-game/assets/79759607/f3fc90b1-e142-4dd1-8f1d-9a8f59f64323)

## Features
- Interactive Tic-Tac-Toe game board
- Track player turns (Player X and Player O)
- Customize Player Names
- Highlights the winner when the game is decided
- Option to reset the game after it ends

## Technologies Used
- React.js
- JSX
- useState
- CSS for styling
## Installation

To run this project locally, you'll need to have Node.js installed. Follow these steps:

- Clone the repository:

```bash
  git clone https://github.com/deepfuriya/react-tic-tac-toe-game
```

- Navigate to the project directory:

```bash
  cd react-tic-tac-toe-game
```

- Install dependencies:

```bash
  npm install
```

- Start the development server:

```bash
  npm start
```
